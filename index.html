About Me

👨‍💻

Building Secure Infrastructure, One System at a Time

I'm a passionate technologist with expertise spanning cybersecurity, DevOps, and infrastructure automation. By day, I work as a Security Engineer, designing secure systems, implementing security controls, and ensuring robust protection across the entire technology stack. By night, I continue exploring new technologies and security frameworks through personal projects and research.

My journey in tech has taken me through various cybersecurity roles where I've gained deep experience in threat detection, vulnerability assessment, and security architecture. This security-first mindset now influences everything I build, ensuring that every system I design prioritizes security without compromising functionality.

I'm particularly enthusiastic about homelab setups and self-hosted solutions. I've built my own secure infrastructure including a personal password vault, comprehensive home automation system, and secure remote access using reverse proxies with Cloudflare Access for robust user authentication. These projects allow me to experiment with enterprise-grade security concepts in a controlled environment while maintaining complete control over my data and privacy.
